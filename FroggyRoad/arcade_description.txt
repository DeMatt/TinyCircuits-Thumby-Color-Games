FroggyRoad

A clone of the Atari game Frogger, but it's infinite like Crossy Road. It also has re-playable worlds.
Don't get hit by cars, or drown in water!

Menu Controls:
  - A button starts the game
  - Up/Down on +-pad to select world number
  - Right on +-pad to select last world (#99)
  - Left on +-pad to go to select first world (#1)

Game Controls:
  - Up or Right Bumper to move forward
  - Right/Left to move thier respective directions
  - Menu button ends the game and returns to menu
